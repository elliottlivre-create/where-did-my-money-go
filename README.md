# Vart tog mina pengar vägen? (Vanilla JavaScript)

Ett simpelt system som skapar en lättare väg för andvändaren att kunna 
se och förstå vart sina pengar tar vägen. 

## Features 
- Visa total spending 
- Expenses are saved using localstorage 
- Add expenses with amount, category and note

## Tech stack 
- HTML
- CSS
- Javascript (vanilla)

## Purpose 
- Andvända själv
- Protfolie projekt 

## Status 
- In progress

- ## Deploy
- - För att lätt köra systemet ladda ner alla filer, och kör i terminalen "open index.html"
